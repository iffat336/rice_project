## Related Research Themes

- Rice heat stress.
- Transcriptome analysis.
- Alternative splicing.
- Temperature memory.
- Post-transcriptional regulation.
- Candidate-gene validation for thermotolerance.

## How This Project Uses Those Themes

This project proposes a computational bridge between transcriptomic evidence and biological validation. It is designed to be realistic for a PhD discussion: start with public or lab-generated RNA-seq data, detect heat-responsive expression and splicing events, then rank candidates for follow-up.
